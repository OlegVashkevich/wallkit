# Компонент Checkbox

Базовый компонент флажка (чекбокса). Реализует стандартный HTML-элемент `<input type="checkbox">` с полной типобезопасностью на PHP и кастомизацией через CSS-переменные.

## Характеристики

| Параметр | Значение |
|----------|----------|
| **Тип компонента** | Базовый элемент формы |
| **Иммутабельность** | Да (readonly) |
| **Рендеринг** | SSR (серверный) |
| **Типизация** | Strict PHP 8.2+ |
| **CSS префикс** | `wallkit-` |
| **Зависимости** | Base, WithHelpers, WithStrictHelpers, WithInheritance |

## Поддерживаемые типы использования

| Тип | Назначение | Особенности |
|-----|------------|-------------|
| **Одиночный чекбокс** | Флажок для одного значения | Состояние true/false |
| **Группа чекбоксов** | Несколько значений с общим именем | Массив выбранных значений |
| **Обязательный выбор** | Чекбокс с валидацией | required атрибут |
| **Состояния** | Включен/выключен/отключен | checked/disabled |

## Свойства компонента

| Свойство | Тип | По умолчанию | Описание |
|----------|-----|--------------|----------|
| `name` | `string` | — | Имя поля (обязательно) |
| `value` | `?string` | `'on'` | Значение чекбокса при отправке |
| `label` | `?string` | `null` | Текст подписи к чекбоксу |
| `checked` | `bool` | `false` | Выбран ли чекбокс изначально |
| `required` | `bool` | `false` | Обязательный ли выбор |
| `disabled` | `bool` | `false` | Отключенное состояние |
| `id` | `?string` | `null` | HTML ID (автогенерация) |

## Использование

### Базовое использование
```php
use OlegV\WallKit\Form\Checkbox\Checkbox;

$checkbox = new Checkbox(
    name: 'accept_terms',
    label: 'Принимаю условия соглашения'
);
echo $checkbox;
```

### Предвыбранный чекбокс
```php
$newsletter = new Checkbox(
    name: 'newsletter',
    label: 'Подписаться на рассылку',
    checked: true
);
```

### Группа чекбоксов
```php
$hobbies = [
    new Checkbox(name: 'hobbies', value: 'reading', label: 'Чтение'),
    new Checkbox(name: 'hobbies', value: 'sports', label: 'Спорт'),
    new Checkbox(name: 'hobbies', value: 'music', label: 'Музыка'),
];
```

### Обязательный чекбокс
```php
$requiredCheckbox = new Checkbox(
    name: 'privacy_policy',
    label: 'Согласен с политикой конфиденциальности',
    required: true
);
```

## Стилизация

### CSS-классы
| Класс | Назначение |
|-------|------------|
| `.wallkit-checkbox` | Базовый контейнер чекбокса |
| `.wallkit-checkbox__input` | Скрытый нативный input |
| `.wallkit-checkbox__label` | Подпись к чекбоксу |
| `.wallkit-checkbox--disabled` | Отключенное состояние |

### CSS-переменные
Компонент использует переменные из базовой темы:
- `--wk-color-*` — цвета
- `--wk-spacing-*` — отступы
- `--wk-font-*` — типографика
- `--wk-border-radius-*` — радиусы скругления
- `--wk-shadow-*` — тени
- `--wk-transition-*` — анимации

### Кастомизация внешнего вида
Компонент использует кастомное оформление с помощью CSS:
- Скрытый нативный чекбокс
- Кастомный квадрат с галочкой
- Плавные переходы состояний
- Состояния: hover, focus, checked, disabled

## Доступность (A11y)

Компонент обеспечивает:
- Правильную связь `label` с `input` через атрибут `for`
- Клавиатурную навигацию (Tab, Space для выбора)
- Состояния фокуса для пользователей клавиатуры
- Адекватные цветовые контрасты
