# WallKit

Библиотека компонентов для построения пользовательских интерфейсов на PHP.

## О проекте

WallKit — это современная библиотека UI-компонентов для PHP, построенная на принципах иммутабельности, производительности и простоты использования. Каждый компонент представляет собой независимый, самодостаточный блок с собственными стилями и логикой.

## Особенности

- **Иммутабельные компоненты** — предсказуемый рендеринг и безопасность
- **Высокая производительность** — кэширование на всех уровнях
- **Кастомизация через CSS-переменные** — легкая настройка внешнего вида
- **Доступность** — встроенная поддержка ARIA и семантики
- **Гибкость** — трейты для расширения функциональности
- **Полное тестирование** — юнит, интеграционные и E2E тесты

## Установка

```bash
composer require olegv/wallkit
```

## Быстрый старт

```php
<?php
require __DIR__.'/vendor/autoload.php';

use OlegV\WallKit\Base\Base;
use OlegV\WallKit\Form\Input\Input;
use OlegV\BrickManager;

// Создание простого компонента
$input = new Input(
    name: 'username',
    label: 'Имя пользователя',
    placeholder: 'Введите имя'
);

echo $input;
echo BrickManager::getInstance()->renderAssets();
```

## Архитектура

Библиотека построена на базе `Brick` — иммутабельного класса компонентов, который обеспечивает:

```php
// Каждый компонент состоит из 4 файлов:
// 1. Input.php        - PHP класс
// 2. template.php     - HTML шаблон
// 3. style.css        - Стили
// 4. script.js        - JavaScript (опционально)
```

### BrickManager
Центральный менеджер для управления всеми компонентами:
- Кэширование компонентов
- Управление ассетами (CSS/JS)
- Регистрация и рендеринг

## Кастомизация

Все компоненты используют CSS-переменные для настройки:

```css
:root {
    /* Акцентные цвета */
    --wk-color-primary: #4a6fa5;
    --wk-color-secondary: #6b7280;
    
    /* Размеры */
    --wk-spacing-2: 0.5rem;
    --wk-radius-md: 0.375rem;
    
    /* Шрифты */
    --wk-font-family: -apple-system, sans-serif;
}
```

Переопределите переменные в своем проекте для настройки внешнего вида.

## Расширение

Используйте трейты для добавления функциональности:

- `WithCache` — кэширование рендера
- `WithHelpers` — вспомогательные методы для шаблонов
- `WithInheritance` — наследование стилей и шаблонов
- `WithStrictHelpers` — строгие проверки типов
